# Calculator App
**This Repo was entirely generated in one shot by a team of AI Agents connected with GitHubs MCP server.**



Monorepo with an elegant React frontend and a functional Node/Express backend.

- Frontend: React + Vite + TypeScript + Tailwind CSS
- Backend: Node.js + Express + TypeScript

## Getting started

Requirements: Node.js 20 LTS and npm

Install dependencies for all workspaces:

```
npm run install:all
```

Run frontend (Vite dev server on http://localhost:5173):

```
npm run dev:frontend
```

Run backend (Express dev server on http://localhost:3000):

```
npm run dev:backend
```

## Repository structure

- `/frontend` — React app
- `/backend` — Express API
- `/.github` — GitHub workflows (to be added)
